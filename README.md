🍔 Sistema de Gerenciamento - Hamburgueria

Este é um projeto em Python desenvolvido para simular o funcionamento de uma hamburgueria, permitindo ao usuário interagir com um sistema de pedidos.

O programa utiliza estruturas básicas da linguagem para gerenciar cadastro de usuário, seleção de itens, pedidos e entrega.

🚀 Funcionalidades

O sistema permite:

Cadastro de Conta: O usuário pode cadastrar um email e senha (simulando conta Google).
Seleção de Itens: Escolha de produtos do cardápio como hambúrguer, batata frita e refrigerante.
Finalização de Pedido: Simulação do preparo do pedido.
Rastreamento: Exibição do status do pedido.
Cadastro de Endereço: Inserção do endereço para entrega.
Entrega: Atualização do status para “saiu para entrega” e “entregue”.
Encerramento: Mensagem de agradecimento ao cliente.
🛠️ Tecnologias e Estruturas Utilizadas

O projeto foi construído utilizando conceitos fundamentais de Python, sem a necessidade de bibliotecas externas:

Listas ([]): Gestão da fila e histórico cronológico.
Dicionários ({}): Mapeamento de serviços e preços.
Sets ({}): Filtro de serviços únicos para relatórios.
Laços de Repetição (while): Menu interativo e persistente.
Condicionais (if/elif/else): Lógica de tomada de decisão.
